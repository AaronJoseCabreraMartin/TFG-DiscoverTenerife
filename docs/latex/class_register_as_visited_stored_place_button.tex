\begin{DoxyItemize}

\item If the user is close enough but he didnt waited the enough time, it will show a toast with the following message\+: \char`\"{}\+You have to wait to register this place as visited again\char`\"{}.
\item If the user is close enough and the cooldown time has finished, it counts the new visit to the current stored chosen place and make a toast wiht the following message\+: \char`\"{}\+You register this place as visited successfully\char`\"{}.
