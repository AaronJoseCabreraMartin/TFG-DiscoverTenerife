This method register a visit of the current user to the place that has the given type and id. It registers the visit at the given timestamp. If the user havent visit a place with the given type and id this method adds a new \mbox{\hyperlink{class_visited_place}{Visited\+Place}} object to the visited\+Places\+\_\+ array. It also adds to the user a bonus of score. The obtained score depends on\+: if its the first time that the user visit that place and the proportion of visits that place has compared to the most visited place. It is calculated using get\+Score\+For\+Visiting\+APlace method of \mbox{\hyperlink{classgame_rules}{game\+Rules}} class. It also checks if one of the challenges was visit that place, if that is the case it use the calculate\+Challenge\+Score method of \mbox{\hyperlink{classgame_rules}{game\+Rules}} class.
