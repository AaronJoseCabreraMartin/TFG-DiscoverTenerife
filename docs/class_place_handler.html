Detailed Description

This class handles an element of the site panel. It shows a loading gif animation when its downloading the image of the showed place.

◆ Update()

void PlaceHandler.Update ( )
private

This method is called once per frame.

  • It checks if the serverHandler_ static property is null, it that is the case it tries to get the firebaseHandlerInstance_ static propert of firebaseHandler.
  • If the serverHandler static property isnt null, and the place didnt load al ready, if its the turn of the current place, it starts the download and add 1 to the turn. And if the place has been selected but they are ready, it calls the loadPlace method.
